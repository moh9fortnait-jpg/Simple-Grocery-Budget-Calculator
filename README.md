# Project 3: Simple Grocery Budget Calculator

A basic Python command-line utility designed to help families track their daily grocery shopping budget. It calculates the total cost of three essential items and compares it with the available budget.

## How it Works:
1. The user inputs their total available budget (in local currency).
2. The user enters the prices of three different shopping items.
3. The program calculates the total sum and checks if the budget covers it.
4. It displays the remaining money or the missing amount.

## Concepts Used:
- Numeric inputs and type casting (`float()`)
- Basic arithmetic (Addition and Subtraction)
- Conditional evaluation (`if-else`)
